## Building

### Dependencies

You need to have **blueprint-compiler** installed on your system.

Fedora

```
sudo dnf install blueprint-compiler
```

Ubuntu & Debian

```
sudo apt install blueprint-compiler
```

To run this project, run the following command:

```bash
dotnet run
```
